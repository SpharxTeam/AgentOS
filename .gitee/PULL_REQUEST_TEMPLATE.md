# Pull Request 模板

感谢您为 AgentOS 项目贡献代码！在提交 PR 之前，请确保您已经完成以下检查：

## 检查清单

### 代码质量
- [ ] 代码遵循项目的编码规范（C/C++/Python 等）
- [ ] 代码已通过 clang-format/clang-tidy 格式化检查
- [ ] 新增的公共 API 都有完整的 Doxygen 注释
- [ ] 代码没有引入新的编译器警告

### 测试
- [ ] 新增代码有相应的单元测试
- [ ] 所有现有测试仍然通过
- [ ] 测试覆盖率没有下降
- [ ] 如果是性能优化，请提供基准测试结果

### 文档
- [ ] 更新了相关的文档（README、API 文档等）
- [ ] 新增的功能有使用示例或教程
- [ ] 更新了 CHANGELOG.md（如果适用）

### 其他
- [ ] PR 描述清晰，说明了变更的原因和内容
- [ ] 关联了相关的 Issue（格式：Closes #123）
- [ ] 分支名称符合规范（feature/xxx, bugfix/xxx 等）

## 注意事项

1. **代码审查**：维护者可能会提出修改建议，请及时响应
2. **合并策略**：功能分支使用 squash and merge，bug 修复使用 merge commit
3. **许可证**：确认您的贡献符合项目的开源协议
