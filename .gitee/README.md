# AgentOS Gitee 配置目录

本目录包含 AgentOS 项目在 Gitee 平台上的特定配置文件。

## 目录结构

- `workflows/` - Gitee Go 流水线配置
- `ISSUE_TEMPLATE/` - Issue 模板
- `PULL_REQUEST_TEMPLATE.md` - Pull Request 模板
- `CODEOWNERS` - 代码所有者文件

## 关于 Gitee 平台

AgentOS 的主仓库托管在 Gitee：https://gitee.com/spharx/agentos

## 配置说明

### 1. Gitee Go 流水线

Gitee Go 是 Gitee 提供的 CI/CD 服务，类似于 GitHub Actions。我们提供了跨平台的构建和测试流水线。

### 2. Issue 模板

为了规范 Issue 提交，我们提供了多种 Issue 模板：
- Bug 报告
- 功能建议
- 文档改进
- 安全问题

### 3. Pull Request 模板

PR 模板帮助贡献者提供必要的信息，便于代码审查。

### 4. 代码所有者

CODEOWNERS 文件定义了各个模块的代码负责人，确保代码变更得到合适的审查。
