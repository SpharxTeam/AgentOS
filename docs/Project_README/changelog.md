## [1.0.0] - 2026-04-05

### 🎉 新增 (Added)

#### 核心架构
- ✨ 实现**五维正交设计原则**完整体系（24条原则）
- ✨ **微内核架构** (CoreKern): IPC/Mem/Task/Time 四大原子机制 (~25K LOC)
- ✨ **三层认知循环** (CoreLoopThree): Cognition→Planning→Action 架构
- ✨ **四层记忆卷载** (MemoryRovol): L1→L2→L3→L4 分层记忆系统
- ✨ **安全穹顶** (Cupolas): Workbench/Permission/Sanitizer/Audit 四层防护
- ✨ **双系统协同机制**: System 1 (快速直觉) + System 2 (深度推理)
- ✨ **遗忘机制**: 基于艾宾浩斯曲线的记忆权重衰减

#### 守护进程层
- ✨ `llm_d`: LLM服务管理守护进程（支持多提供商路由）
- ✨ `market_d`: 工具市场注册与发现服务
- ✨ `monit_d`: 系统监控与告警服务
- ✨ `sched_d`: 任务调度器（支持优先级队列和多算法）
- ✨ `tool_d`: 工具执行引擎（沙箱隔离）
- ✨ `gateway_d`: API网关（限流、认证、CORS）

#### 多语言SDK
- ✨ **Python SDK**: 完整的类型注解API客户端
- ✨ **Go SDK**: 并发安全的API客户端
- ✨ **Rust SDK**: 零成本抽象的高性能绑定
- ✨ **TypeScript SDK**: 支持Node.js和浏览器环境

#### 应用层
- ✨ **OpenLab Web UI**: Vue 3 + TypeScript 构建的交互平台
- ✨ **CLI工具**: 命令行管理工具 (`agentos-cli`)
- ✨ **RESTful API**: 完整的内核API（符合OpenAPI 3.0规范）

#### DevOps & 运维
- ✨ **Docker容器化方案**: 多阶段构建、生产级编排
- ✨ **Docker Compose**: 开发环境和生产环境配置
- ✨ **Prometheus + Grafana**: 完整监控栈（含预置仪表盘）
- ✨ **CI/CD流水线**: GitCode/GitHub/Gitee 三平台支持
- ✨ **自动化测试**: 单元测试、集成测试、E2E测试框架

#### 文档体系
- ✨ **完整文档中心** (docs/): 33篇文档，约165,000字
- ✨ **架构设计文档**: 五维正交24条原则详解
- ✨ **API参考文档**: 所有接口的完整说明与示例
- ✨ **开发者指南**: 编码规范、贡献指南、调试技巧
- ✨ **运维手册**: 部署、监控、备份恢复、故障排查

#### 平台工程
- ✨ **CODEOWNERS**: 代码所有权声明（自动PR审查分配）
- ✨ **SECURITY.md**: 安全披露政策与响应流程
- ✨ **CONTRIBUTING.md**: 完整的贡献者指南
- ✨ **PR模板**: 双语（中英文）Pull Request模板

## 版本说明

### 版本号规则

AgentOS遵循**语义化版本 2.0.0**:

```
MAJOR.MINOR.PATCH

MAJOR: 不兼容的API变更
MINOR: 向后兼容的功能新增
PATCH: 向后兼容的问题修复
```

### 预发布标识

- `alpha`: 内部测试版，功能可能不稳定
- `beta': 公开测试版，功能基本稳定，欢迎反馈
- `rc`: 发布候选版，除非发现严重Bug否则不再改动

示例: `1.0.0-beta.2`, `2.0.0-alpha.1`, `1.1.0-rc.1`
