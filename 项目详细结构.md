# AgentOS 项目完整目录结构

本文档详细描述了 AgentOS 项目的完整目录结构，包含所有文件层级和关键说明。

---

## 📁 根目录 (AgentOS/)

### 顶层配置文件

| 文件/目录 | 说明 |
|-----------|------|
| **README.md** | 项目主文档，包含系统架构、核心功能、快速开始指南等 |
| **CHANGELOG.md** | 变更日志，记录所有版本的功能新增、改进和 Bug 修复 |
| **LICENSE** | Apache License 2.0 开源协议文本 |
| **NOTICE** | 版权和第三方组件声明 |
| **pyproject.toml** | Python 项目配置文件（Poetry），定义依赖、脚本和工具配置 |
| **Makefile** | 构建自动化脚本，提供 build、test、clean 等常用命令 |
| **.gitignore** | Git 忽略规则，排除编译产物、缓存、敏感文件等 |
| **.editorconfig** | 编辑器统一配置，确保跨编辑器的代码风格一致 |
| **.pre-commit-config.yaml** | Pre-commit 钩子配置，自动执行代码格式化、检查等 |
| **.env.example** | 环境变量模板文件，包含配置项示例 |
| **quickstart.sh** | 一键快速启动脚本 |
| **validate.sh** | 环境验证脚本，检查项目结构和依赖完整性 |

### 核心模块目录

| 目录 | 说明 |
|------|------|
| **atoms/** | 微内核层 - CoreLoopThree 三层一体架构的核心实现 |
| **backs/** | 核心服务层 - LLM、工具、市场、调度、监控等服务守护进程 |
| **domes/** | 安全隔离层 - 权限控制、审计日志、沙箱机制 |
| **dynamic/** | 运行时服务 - HTTP/WebSocket网关、会话管理、遥测服务 |
| **config/** | 配置文件中心 - Agent、内核、日志、模型、安全等配置 |
| **tools/** | 多语言 SDK - Go、Python、Rust、TypeScript 接口封装 |
| **openhub/** | 开放协作中心 - 应用示例、社区贡献、市场机制 |
| **partdata/** | 数据分区 - 内核数据、日志、追踪、注册表等 |
| **partdocs/** | 技术文档中心 - 架构设计、开发指南、技术规范 |
| **scripts/** | 运维脚本工具 - 初始化、构建、文档生成、基准测试等 |
| **tests/** | 测试套件 - 单元测试、集成测试、安全测试、性能基准测试 |

## 📦 atoms/ - 微内核层

**定位**: CoreLoopThree 三层一体架构的核心实现，包含认知层、行动层、记忆层的完整运行时

```

│   ├── README.md                  # 微内核设计文档
│   ├── CMakeLists.txt             # 构建配置
```
