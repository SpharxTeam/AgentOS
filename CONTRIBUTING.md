# 贡献指南

感谢您对 AgentOS 的兴趣！请遵循以下指南。

## 报告问题

- 使用 GitHub Issues 报告 bug 或功能请求。
- 请提供清晰的描述和重现步骤。

## 提交代码

1. Fork 本仓库。
2. 创建新分支 (`git checkout -b feature/your-feature`)。
3. 确保代码通过所有测试 (`pytest tests/`)。
4. 确保契约验证通过 (`python scripts/validate_contracts.py`)。
5. 提交 PR，描述您的更改。

## 代码风格

- 遵循 PEP8。
- 所有文件必须包含版权声明。
- 函数和类需有 docstring。

## 许可证

贡献的代码将采用 GPL-3.0 许可证。
