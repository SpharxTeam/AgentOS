## 🔧 维护指南 / Maintenance Guidelines

### 更新主文档 / Updating Master Document
1. 首先更新简体中文版本 (作为权威参考)
2. 同步更新英文版本
3. 根据需要更新法文和德文精简版

First update Simplified Chinese version (as authoritative reference), then synchronize English version, finally update French and German concise versions as needed.

Key terminology should maintain cross-language consistency:
